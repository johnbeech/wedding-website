# Hannah and John's Wedding 2019 Website

Our wedding website.

## Requirements

- PHP >= 7.#.#
- Node JS >= 8 LTS

## Development

From this directory:

- Download composer: `curl -s https://raw.githubusercontent.com/composer/getcomposer.org/master/web/installer | php`
- Run `php composer.phar install`
- Run `npm install`
- Run `npm start`
